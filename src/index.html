My Work

  • Chess Algo

    This implements a breadth-first search of an adjacency graph to find a knight's shortest path from A to B on *any size chess board.

  • Weather App

    This makes asynchronous calls to WeatherAPI.com to get real-time weather data, and saves a user's locations / updates them upon page load.

  • Battleship

    This was more of a demo highlighting TDD methodology by implementing Jest - unit test suites were written for every public function/class.

  • To-do App

    This is a multi-project, configurable to-do list app highlighting OOP. Lists and their respective titles/descriptions dynamically update and can be saved to localStorage.

  • Tic Tac Toe

    Plain, simple and sleek Tic Tac Toe experience. Makes use of the Pub/Sub model with vanilla JS to get data to all the UI components. Multi-player functionality.

  • UI Demos

    A demonstration of several interactive UI components. Includes use of transitions for menus and side-bar, full-featured form validator and image slider.
